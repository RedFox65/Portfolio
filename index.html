Le jeu du Sokoban

Projet d'études - BUT 1 (2025-2026)

Description

Dans ce jeu, un personnage doit déplacer des caisses pour les ranger sur des positions cibles. Il peut se déplacer dans les quatre directions et pousser une seule caisse à la fois. L’objectif pour le joueur est de réussir à positionner toutes les caisses sur les cibles avec le moins de déplacements et de poussées possibles.

Objectif du projet

Développer le jeu de réflexion Sokoban en langage C. L'objectif étant d'appliquer nos connaissances dans ce langage de programmation, en algorithmique et en gestion de la mémoire.

Technologies utilisées

  • C
  • Git

Défi technique

Problème : La gestion de l'historique des coups pour permettre au joueur d'annuler une action (Ctrl+Z) consommait trop de mémoire.

Solution : J'ai donc implémenté une structure de données en liste chaînée pour stocker uniquement les différences entre chaque déplacement plutôt que de sauvegarder toute la grille.

Comparaison Algorithmiques

L’objectif de rendre autonome et intelligent le Sokoban en lui fournissant un plateau et une suite de déplacements à effectuer.

Détails

Comparaison Algorithmiques

Projet d'études - BUT 1 (2025-2026)

Description

Dans sa première version, le programme du jeu Sokoban permettait à un joueur de pouvoir jouer une partie, c’est-à-dire exécuter se déplacer sur le plateau avec les caisses afin de les mettre au bon endroit.

Objectif du projet

Rendre autonome et intelligent le Sokoban en lui fournissant un plateau et une suite de déplacements à effectuer. Il doit aussi détecter si la suite constitue bien une solution et éliminer de la suite de déplacements toute séquence inutile.

Technologies utilisées

  • C
  • Git

Défi technique

Difficulté : Les séquences inutiles étaientt assez difficiles à détecter et à supprimer sans toucher aux déplacements valides.

Solution : Pour cela, il a fallut chercher les combinaisons de déplacements inutiles possibles. Puis ensuite il a fallut trouver des points identiques dans la façons de les résoudres, avant de rédiger l'algorithme qui permet de supprimer ces séquences inutiles.
